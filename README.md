# Mission Control

A data-dense, keyboard-first dashboard for managing Clawdbot agents.

## Features

- **Tasks** - Kanban board for task management
- **Approvals** - Review and approve agent actions
- **Calendar** - Event management
- **Projects** - Project tracking with nested tasks
- **Memory** - Browse agent memory files
- **Docs** - Internal documentation wiki
- **People** - Contact management
- **Cron** - Scheduled job management

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
